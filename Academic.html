Kusitm is a gathering of university students to learn about startups. I was able to learn a lot from weekly sessions and internal team events.

Competition PT


Competition Pit is held by inquiries to the conference at an actual startup company. I was able to learn a lot about collaboration, and it was an opportunity to learn not only about it but also management.

Academic Festival


The academic festival is a place where everything from business plan to implementation is evaluated. It was an opportunity to use everything I learned during in Kusitm.

System


It uses data from various sensors such as LIDAR, IMU, camera, and GPS. Driven using collected data and open source 'autoware'. Most of the time was spent implementing drivers for data processing and operation, and optimization through testing.

PID


PID stands for Proportional-Integral-Differential controller and acts as a feedback control for vehicle driving. PID performance depends on how quickly and accurately the target is reached.

Path Following


There are geometric and kinematic methods for path following. It is a complex but kinematic method that allows more precise driving. In path tracking, most of the investment was invested in finding an optimized parameter.
